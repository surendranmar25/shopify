# Shopify Price Updater

## Setup

1. Install Python
2. Open this folder in VS Code
3. Open terminal
4. Run:

pip install -r requirements.txt

5. Update app.py:

SHOP_URL = "yourstore.myshopify.com"
ACCESS_TOKEN = "shpat_your_token_here"

6. Run app:

streamlit run app.py

## Excel Format

ERP Code | Display Price

Example:
LAPTOP001 | 54999
MOUSE002 | 999
